## Instalação

1. Clone o repositório:

   ```bash
   git clone git@github.com:leofrs/store.git
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Inicie o servidor:

   ```bash
   npm run dev
   ```
